Desktop Bridge — Optional Power Mode

The bridge turns your Mac into a remote-accessible node in the Favour Economy network. Voice-control your computer from your phone. Fully local — nothing leaves your machine.

# Install the bridge server
$ cd ~/FavourEconomy/bridge
$ pip install -r requirements.txt
$ python server.py

# aiButler Bridge running at http://127.0.0.1:8765
# Pair your phone with the token shown above
# LAN mode: AIBUTLER_BRIDGE_ALLOW_LAN=1 python server.py
